# Promotion - Partners Module - UI Component Guide

## Partner Management Flow

### Managing Partners
1. Admin navigates to Promotional Partners page
2. Views list of partners with status badges
3. Can filter by status, date range, and search
4. Can perform actions: View, Edit, Delete
5. Can copy referral codes
6. Can export partner list to CSV

### Creating a Partner
1. Admin clicks "Add Partner" button
2. Fills form with partner details
3. Clicks "Save" button
4. System creates partner with auto-generated referral code
5. Redirects to partners list with success message

### Viewing a Partner
1. Admin clicks View icon on partner row
2. View page displays partner info and performance stats
3. Shows associated promotional posts
4. Can Edit or Delete from view page

---

## API Endpoints

### Partner Management
- `GET /internal/promotion/partners` - Get all partners (with filters)
- `GET /internal/promotion/partners/:id` - Get partner details
- `POST /internal/promotion/partners` - Create new partner
- `PUT /internal/promotion/partners/:id` - Update partner
- `DELETE /internal/promotion/partners/:id` - Delete partner
- `PATCH /internal/promotion/partners/:id/toggle-status` - Toggle partner status

### Posts
- `GET /internal/promotion/partners/:partnerId/posts` - Get partner's posts

## Partners List Page (`/admin/promotion/partners`)

### Page Header
- **Title**: "Promotional Partners"
- **Subtitle**: "Manage referral partners and their performance"
- **Button**: "Add Partner" (green gradient)

### Filters Section
- **Search Input**: Search by name, mobile, email
- **Status Dropdown**: All Status/Active/Inactive/Suspended/Draft
- **Date From**: Date picker
- **Date To**: Date picker
- **Apply Button**: Blue gradient

### Partners Table

**Columns:**
- # (Serial number)
- Created At
- Name
- Mobile
- Referral Code (with copy button)
- Status (badge with icon)
- Actions

**Action Buttons (per row):**
- View (eye icon)
- Edit (pencil icon)
- Delete (trash icon)

**Status Badge:**
- Active: Green background with check icon
- Inactive: Gray background with X icon
- Suspended: Red background with X icon
- Draft: Red background with X icon

### Pagination
- Previous button (◄)
- Page indicator (X / Y)
- Next button (►)
- Results count display

## Create Partner Page (`/admin/promotion/partners/create`)

### Page Header
- **Title**: "Create New Partner"
- **Subtitle**: "Add a new promotional partner"

### Form Section

**Section Title**: "Personal Information"

**Fields:**
- Full Name (required, text input)
- Email Address (required, email input)
- Mobile Number (required, text input)
- Commission Rate (%) (optional, number input with decimals)

**Validation:**
- Name: Minimum 3 characters
- Email: Valid email format
- Mobile: Minimum 7 digits
- Commission Rate: Decimal number

**Form Actions:**
- Cancel button (gray)
- Save button (green gradient)

## View Partner Page (`/admin/promotion/partners/:id`)

### Page Header
- **Back Button**: "Back to Partners"
- **Action Buttons**:
  - Edit (blue gradient)
  - Delete (red gradient)

### Layout (3 columns)

#### Left Column (1/3 width) - Personal Info Card

**Fields Displayed:**
- Name
- Email (with mailto link)
- Mobile (with tel link)
- Referral Code (with copy button)
- Status (uppercase)
- Commission Rate (if available)

#### Right Column (2/3 width) - Performance Summary Card

**Performance Metrics (3 columns):**
- Total Clicks (large number display)
- Conversions (large number display)
- Conversion Rate (percentage, large number display)

**Additional Metrics (2 columns):**
- Total Commission (amount display)
- Joined Date + Last Updated timestamp

### Associated Posts Section

**Table:**
- **Title**: "Associated Promotional Posts (count)"
- **Button**: "View All ►"

**Columns:**
- Post Title
- Type (post_platform)
- Status (Active/Inactive)

**Empty State**: "No posts"

### Recent Activity Section

**Title**: "Recent Activity"
**Content**: "No activity data available."

## Features

### Copy Referral Code
- Click clipboard icon next to referral code
- Copies code to clipboard
- No visual feedback (silent copy)

### Export to CSV
- Exports all partners in current view
- Columns: #, Name, Mobile, Email, Referral Code, Status, Active
- Filename: `partners.csv`
- Handles special characters and quotes

### Delete Confirmation
- Shows browser confirm dialog
- Message: "Delete this partner? This action cannot be undone."
- Deletes partner and refreshes list on confirmation

---

## Notes
- Referral code is auto-generated by backend
- Status can be: active, inactive, suspended, draft
- Commission rate is optional and stored as decimal
- Performance stats come from analytics API
- Partners can have multiple associated posts
- CSV export uses current filter/search criteria
- All monetary values displayed without currency symbol
- Dates formatted using browser's locale
